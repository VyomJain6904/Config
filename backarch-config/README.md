# Installation Script :

### Install the Windows Chicago 95 Theme :

```zsh
git clone https://github.com/grassmunk/Chicago95.git
```

```zsh
cd Chicago95
chmod +x installer.py
python 3 installer.py
```
---

### Copy the Wallpaper from the Github :

```zsh
https://github.com/VyomJain6904/Config
cd ~/Config/blackarch-config/wallpaper/
cp -r w.jpg w1.jpg ~/Pictures/ 
```

### Copy the ZSHRC Config :

```zsh
cd ~/Config/backarch-config/.zshrc
```

### Copy the Config :

```zsh
cd ~/Config/backarch-config/ 
cp -r nvim terminator ~/.config/ 
```

### Install Font :

```zsh
sudo pacman -S ttf-jetbrains-mono-nerd
```
